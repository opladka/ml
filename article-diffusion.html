The Mathematics of Diffusion Models

CW
Chen Wei
May 22, 2026
11 min read
Papers

Generative AI was long dominated by Generative Adversarial Networks (GANs) and Variational Autoencoders (VAEs). While GANs could produce sharp images, their training loop was notoriously unstable and subject to mode collapse. Variational models offered stable training, but outputs were often blurry.

Diffusion Models bypassed these limitations by framing generation as a physical thermodynamic process: systematically adding noise to data, then learning to reverse the noise addition step-by-step.

The Forward Process (Diffusion)

Given a clean data point x0, the forward process adds Gaussian noise in T steps. The step transition probability is governed by a variance schedule βt:

q(xt | xt-1) = N( xt; √(1 - βt) xt-1, βt I )

Using the reparameterization trick, we can jump directly from x0 to any arbitrary time step t:

xt = √ᾱt x0 + √(1 - ᾱt) ε

Where αt = 1 - βt and ᾱt = ∏i=1t αi. When T is large, xT becomes indistinguishable from pure isotropic Gaussian noise.

The Reverse Process (Generation)

If we can estimate the reverse transitions pθ(xt-1 | xt), we can start from noise xT and generate a clean sample. Since the true reverse distribution is intractable, we train a neural network (typically a U-Net) to predict the added noise vector ε.

The simplified loss objective proposed by Ho et al. (2020) minimizes the L2 distance between the actual added noise and the network's prediction:

Lsimple(θ) = Et, x0, ε [ || ε - εθ( √ᾱt x0 + √(1 - ᾱt) ε, t ) ||2 ]
DenoiseStep.py (JAX / Equinox snippet) 
import jax
import jax.numpy as jnp

def reverse_step(model, x_t, t, beta_t, alpha_bar_t, key):
    # Predict noise using U-Net
    pred_noise = model(x_t, t)
    
    # Calculate mean reconstruction
    coef = beta_t / jnp.sqrt(1 - alpha_bar_t)
    mean = (x_t - coef * pred_noise) / jnp.sqrt(1 - beta_t)
    
    # Add Langevin noise
    z = jax.random.normal(key, shape=x_t.shape)
    sigma_t = jnp.sqrt(beta_t)
    
    return mean + sigma_t * z
